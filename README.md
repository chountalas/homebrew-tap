# Connor Hountalas Homebrew Tap

Homebrew formulae and casks for Connor Hountalas apps.

## Redline

Install everything: the native Mac app, `redline`, `redline-mcp`, and the provider adapters for Codex, Ollama, OpenAI, and Anthropic.

```bash
brew install --cask chountalas/tap/redline-app
```

CLI and MCP tools only:

```bash
brew install chountalas/tap/redline
```

Upgrade later with:

```bash
brew upgrade chountalas/tap/redline
brew upgrade --cask chountalas/tap/redline-app
```

brew install --cask chountalas/tap/luma
